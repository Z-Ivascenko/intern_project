# Projekta kopsavilkums

Projekta laikā izveidoju pamata backend lietotni, izmantojot Node.js un Express.  
Konfigurēju MySQL datubāzes savienojumu un izveidoju lietotāju tabulu ar validāciju un unikālu e-pasta lauku.  
Realizēju REST API maršrutus lietotāju izveidei un iegūšanai, tai skaitā filtrēšanu un lapošanu.  
Ieviesu datu validāciju, paroļu hešošanu un vienotu kļūdu apstrādi.  
Pārbaudīju risinājumu ar Postman, kā arī izveidoju vienību testus ar Jest un API testus ar Supertest.  
Noslēgumā sakārtoju projekta struktūru, veicu nelielu refaktorēšanu un sagatavoju dokumentāciju.

## Programmu projektēšana (3. nodaļa) – kopsavilkums

Projektēšanas posmā apkopoju sistēmas prasības un uzrakstīju lietotāja stāstus, kā arī sagatavoju use case sarakstu.  
Izstrādāju augsta līmeņa arhitektūras aprakstu (MVC/REST) un definēju moduļus: routes, controllers, services, db un middleware.  
Sagatavoju API specifikāciju ar galvenajiem galapunktiem un parametriem, definēju datu modeļus (User, Post, Log) un izveidoju ERD diagrammu.  
Aprakstīju validācijas noteikumus un vienotu kļūdu formātu ar kļūdu kodiem, kā arī sagatavoju middleware pseudo-kodu.  
Izveidoju UI/UX skices galvenajām lapām (reģistrācija, ziņas pievienošana, saraksts) un saglabāju tās dokumentācijā.
